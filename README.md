Эта программа решает судоку. При этом находит даже несколько вариантов решений (до 9), если они есть!
Входной файл с игровыми значениями задаётся в текстовом формате. Путь к этому файлу передаётся в первом параметре при запуске программы.
Пример входного файла можно увидеть в папке games.
Пустые значения задаются в виде одного пробела. Числа записываются без пробелов.
Программа generateCheckList.mjs - генерирует файл checkList.json. Он нужен для работы основной программы.

```
node sudoku.mjs games/game.txt
```
